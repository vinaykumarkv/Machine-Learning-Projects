Welcome to the Certificate Submission API! Below are the available endpoints.

POST /api/submit

Description: Submits certificate data for validation using the ML model.

Request Headers:

Content-Type: application/json

Request Body:

{
  "data": [0.6, 0.8, 0.4]
}

Response:

{
  "status": "success",
  "prediction": "Pass",
  "message": "Certificate processed: Pass"
}

Example PowerShell:

Invoke-WebRequest -Uri "http://127.0.0.1:5000/api/submit" `
  -Method POST `
  -Headers @{ "Content-Type" = "application/json" } `
  -Body '{"data":[0.6,0.8,0.4]}'

Example cURL:

curl -X POST http://127.0.0.1:5000/api/submit \
     -H "Content-Type: application/json" \
     -d '{"data":[0.6,0.8,0.4]}'
